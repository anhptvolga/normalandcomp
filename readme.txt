	NormalizingAndComparingExpressions – программа 
преобразования и проверка эквивалентности двух выражений 
на языке C++.
	Входные данные получаются в параметрах командной строки. 
Сначала указывается имя программы, а затем через пробел имена двух 
текстовых файлов. Каждый текстовой файл содержит выражение на языке С++.
	Пример: php main.php expr1.txt expr2.txt
	Результатом эквивалентности двух выражений является текстовый файл, 
который имеет имя «result.txt».
	Результатом нормализованных деревьев двух выражений является
файлы «treeExp1.gv» и «treeExp2.gv», содержащие описания нормализованных 
деревьев в формате DOT.
	При возникновении ошибки, сообщение об ошибке записывается на экране.
